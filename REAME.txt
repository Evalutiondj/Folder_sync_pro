Các tính năng chính đã thêm
Đồng bộ real-time:

Sử dụng Watchdog để theo dõi thay đổi file

Tự động cập nhật khi file được thêm/sửa/xóa

Đồng bộ 2 chiều:

Cho phép đồng bộ theo cả hai hướng

Bảo toàn dữ liệu ở cả hai thư mục

Lọc file nâng cao:

Lọc theo loại file (ảnh, tài liệu)

Cho phép thêm bộ lọc tùy chỉnh

Bảo mật:

Mã hóa file trước khi đồng bộ

Tạo và quản lý khóa mã hóa

Giao diện nâng cao:

Hệ thống tab tiện lợi

Menu chuyên nghiệp

Thanh trạng thái

Hệ thống logging:

Ghi log vào cả giao diện và file

Phân loại log (info, warning, error)

Tối ưu hiệu năng:

Kiểm tra hash file để tránh copy không cần thiết

Xử lý đa luồng mượt mà

Hướng dẫn sử dụng
Chọn thư mục nguồn và đích

Chọn chế độ đồng bộ:

Mirror: Đồng bộ chính xác, xóa file thừa

Update: Chỉ cập nhật file mới hơn

Add: Chỉ thêm file mới

Strict: Kiểm tra hash để đảm bảo chính xác

Có thể bật các tùy chọn:

Đồng bộ 2 chiều

Mã hóa file

Đồng bộ real-time

Tự động đồng bộ theo chu kỳ

Sử dụng bộ lọc file để chỉ đồng bộ các loại file cần thiết

Xem log chi tiết trong tab chính hoặc file log.txt
